# Simple blogging app

Main page - list of users and their blogs

CRUD for users

CRUD for blogs

CRUD for comments on blogs

Relations:
```
Blogs 1:M Comments
Blogs 1:M Images(urls)
```

| Model | Attributes |
|-------|------------|
| Blog | Title, Description, Likes, Dislikes |
| Comment | Username, Comment Text | 
| Image | url |
